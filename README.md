# Placement-Prediction-Model
Built a machine learning model to predict student placement outcomes using Logistic Regression and data preprocessing techniques.

**Overview**

This project focuses on predicting whether a student will be placed or not based on academic and performance-related features. It demonstrates the end-to-end machine learning workflow from data preprocessing to model evaluation.

**Problem Statement**

Many institutions struggle to identify factors that influence student placements. This project aims to analyze student data and build a predictive model to determine placement outcomes.

**Dataset**

100 rows and 3 features
Includes: Academic performance indicators and placement status

**Tools & Technologies**

Python
Scikit-learn
Pandas, NumPy
Matplotlib

**Methods**

Data Cleaning and Preprocessing
Feature Scaling using StandardScaler
Train-Test Split
Model Building using Logistic Regression
Model Evaluation using Accuracy Score

**Key Insights**

Academic performance plays a key role in placement outcomes
Model successfully classifies placed vs non-placed students
Data preprocessing improves model accuracy

**Output**

Trained Machine Learning Model
Prediction results (Placed / Not Placed)
Visualization of decision boundary

The Logistic Regression model achieved good accuracy in predicting placement outcomes. This project demonstrates the application of machine learning techniques for classification problems.
